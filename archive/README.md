# Archive

Early prototypes and drafts kept for reference. Nothing in this folder is
used by the application.

- `micro_market.py` — zero-dependency prototype of the marketplace using only
  the Python standard library
- `watchdog.py` — local monitoring script from early development (contains
  machine-specific paths)
- `LAUNCH.py` — script that printed an early deployment checklist
- `index.html` — standalone landing page draft (the live pages are served
  from `frontend/templates/`)
- `ORIGINAL_README.md` — the original README, including the business plan and
  roadmap notes
